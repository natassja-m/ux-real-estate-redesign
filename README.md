UX case study redesigning a real estate app to improve usability and reduce friction
#Real Estate App UX Redesign

##Overview
This project is a UX case study focused on improving the user experience of a real estate platform. The goal was to identify usability issues and redesign key interactions to reduce friction and improve accessibility

##Problem
Users experience friction when searching, viewing, and saving properties. 
- Forced sign-on before saving listings
- Inefficient layout between map and listings
- Non-intuative navigation controls
- Extra steps required to view photos
- Poo organization of property details

##Solution
The redesign focused on:
- Allowing users to browse and save listings without immediate sign-in
- Introducing a resizable layout between map and listings
- Improving navigation placement for better usability
- Enabling seamless photo browsing with directional controls
- Reorganizing property information for better readability

##Outcome
The updated design reduces user friction, improves nagivation, and creates a more intuative and user-friendly home search experience.
